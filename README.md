# CppConsoleRPG （むかしのゆうしゃのたびじ）

## 概要
C++を用いて制作したコンソールRPGです。

マップ遷移、戦闘、レベルアップなどのゲームロジックを実装しています。

## 主な機能
- 二次元配列によるマップ管理
- 座標に基づくマップ遷移処理
- ターン制戦闘システム
- 経験値・レベルアップ機能

## 工夫点
- 乱数を用いたダメージ計算および条件分岐による行動制御を実装しました。
- マップ管理で、特定の文字種に対応してギミックを作成しました。
- 特定のマップでは、視界を制限し、ゲーム性の向上を行いました。
- 友人のプレイテストを基にゲームバランスを調整しました。

## 課題と解決
マップ遷移時に座標のずれにより移動できない問題が発生しました。

座標のずれを特定し、遷移先マップとの座標対応を修正することで解決しました。


## 使用技術
- C++
- Visual Studio 2022

## 操作方法
- **WASD**：移動
- **Q**：コマンド
- **space**：会話を進める
