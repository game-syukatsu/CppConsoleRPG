# CppConsoleRPG
# 作品名：むかしのゆうしゃのたびじ

## 種別
自作ゲームプログラム

## 使用技術
C++

## 開発環境
Visual Studio 2022

## 操作方法
- **WASD**：移動
- **Q**：コマンド
- **space**：会話を進める

## 動作環境
Windows11(64bit)

## 開発期間
大学生の時に1ヶ月

## 工夫点
- 迷いの森や暗黒の荒野では視界が狭くなるようにプログラムを書きました。
- また通常コマンドやレベルアップを判定する関数など、実際に友達に遊んでもらい、これが合った方がいいなどの声を聞きながら何度も制作を行いました。
- また昔のような作品にするため、基本的にひらがなを多く使用しています。
